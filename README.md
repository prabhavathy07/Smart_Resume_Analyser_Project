# Smart Resume Analyzer 🚀

## Project Description
Smart Resume Analyzer is a Flask-based web application that analyzes resumes and gives:

- Resume Score
- ATS Score
- Skill Match Percentage
- Required Skills by Job Role
- Missing Skills
- Suggestions to improve resume

---

## Features

✔ Resume Upload (PDF/Text)  
✔ Job Role Selection  
✔ Skill Extraction  
✔ Role-based Matching  
✔ ATS Score Calculation  
✔ Download Report  

---

## Tech Stack

- Python
- Flask
- HTML
- CSS
