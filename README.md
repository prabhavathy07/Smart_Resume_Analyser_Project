# Smart Resume Analyzer AI

An AI-powered Resume Analyzer built using Python and Flask that analyzes resumes, calculates ATS score, identifies missing skills, provides AI suggestions using Google Gemini, generates interview questions, rewrites resumes, and downloads a PDF report.

---

## Features

- Resume Upload (PDF)
- ATS Score Calculation
- Skill Matching
- Missing Skills Detection
- AI Resume Suggestions (Google Gemini)
- AI Resume Rewrite
- Interview Question Generator
- Resume History
- PDF Report Download
- Email Report
- Beautiful Responsive UI
- Live Deployment on Render
